Instrucciones de GIT

  1. Se abre GIT desde la raiz del proyecto
  2. Se ejecuta el comando git init, para iniciar el GIT y poder trabajar
  3. Se ejecuta el comando git add, para agregar los archivos y cambios al repositorio .
  4. Se ejecuta el comando git commit -m "Mensaje del commit", para dejar el comentario de lo que se edito
  5. Se ejecuta el comando git remote add origin URL
  6. Se ejecuta el comando git push -u origin main
